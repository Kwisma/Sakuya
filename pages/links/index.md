## 友链说明

友链为随机顺序展示，希望交换友链的话可以按此格式在评论区留下信息：

```json
{
    "url": "https://yuumi.link",
    "avatar": "https://cdn.yuumi.link/images/settings/favicon.png",
    "name": "Yuumi",
    "blog": "-Yuumi's Blog-",
    "desc": "続きを鳴らそう——",
    "color": "#D69B54"
}
```
